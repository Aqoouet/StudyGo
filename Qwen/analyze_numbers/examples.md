# Примеры работы программы  

## Пример 0  
**Вход:**  
```go  
[1, 2, 1, 3, 2]  
```  
**Выход:**  
```go  
Частота: map[int]int{1: 2, 2: 2, 3: 1}  
Минимум: 1, Максимум: 3, Среднее: 1  
Мода: [1, 2]  
Чётные: [2, 2], Нечётные: [1, 1, 3]  
Диапазоны (размер 2): {"0-2": [1, 1], "3-4": [2, 2, 3]}  
Медиана: 2  
Уникальные: [1, 2, 3]  
Числа выше среднего: [2, 2, 3]  
```
